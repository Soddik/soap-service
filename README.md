# soap-service
Simple CRUD soap service
## Getting started
1. Clone project: `https://github.com/AntiStC/store_app.git`
2. Build with `mvn clean install`
## Developing
### Build with
- Java 17
- Spring boot
- Spring Data JPA
- WSDL
- PostgreSQL
